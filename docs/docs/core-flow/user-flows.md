# User Flows

Foundra supports two primary flows: developer financing and investor participation.

## Developer Flow

1. **Register**: submit company details
2. **ZK-KYC**: complete privacy-preserving verification
3. **Manual KYC**: complete additional verification checks
4. **Submit Project**: provide location, budget, timeline, and documents
5. **AI Grading**: risk oracle generates a grade
6. **Admin Approval**: final review
7. **Vault Created**: ERC-4626 pool initialized
8. **Fundraising**: investors deposit
9. **Receive Funds**: funds released when target reached
10. **Repay**: repayments until completion

## Investor Flow

1. **ZK-KYC**: complete verification
2. **Browse Pools**: filter by risk grade, APY, and tenor
3. **Deposit USDC**: deposit to project vault
4. **Receive Shares**: ERC-4626 vault tokens
5. **Earn Yield**: yield proportional to holdings
6. **Withdraw**: after maturity, completion, or cancellation
