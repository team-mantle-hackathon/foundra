# Security

Foundra includes standard smart contract security controls and a clear audit status.

## Features

| Feature            | Description                                       |
| ------------------ | ------------------------------------------------- |
| Access Control     | Role-based permissions (Admin, Developer, Oracle) |
| Pausable           | Emergency pause functionality                     |
| Reentrancy Guard   | Protection against reentrancy attacks             |
| Rate Limiting      | Prevents abuse                                    |
| Multi-sig (Planned)| Admin controls (future implementation)            |

## Audit Status

This project has not yet been audited. Use at your own risk.
